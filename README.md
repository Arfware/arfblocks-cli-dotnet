# ArfBlocks CLI

"ArfBlocks-cli" is a Dotnet tool that helps developers for ArfBlocks based operations

Available Commands:

- Generate Docs
- Generate Code (Not Completed)

Detailed documentation is in the 'Z-Docs' folder.
